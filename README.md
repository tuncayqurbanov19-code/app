# idmancım.az — Ürün Vizyonu ve Teknik Yol Haritası

Bu repo, **idmancım.az** platformunun ürün kapsamını ve uygulanabilir teknik planını içerir.

## Kurucu
- **Tuncay Qurbanlı**

## Ürün Tipi
- Mobil Uygulama (**iOS + Android**)
- Gelecekte web tabanlı yönetim/kontrol paneli
- Fitness + Doğa Yürüyüşü + Sağlık + Turizm entegre platformu

## Bu repodaki dokümanlar
- `docs/product-brief-tr.md`: Türkçe ürün özeti, özellik listesi ve aşamalı teslimat planı.
- `docs/system-architecture.md`: Önerilen teknik mimari, servis sınırları, veri modeli ve güvenlik yaklaşımı.

## Hızlı başlangıç (planlama)
1. Ürün kapsamını `docs/product-brief-tr.md` üzerinden netleştir.
2. MVP kapsamını dondur (hareket takibi + başarı yolu + temel tur rezervasyonu).
3. Mobil teknoloji seçimi yap (React Native veya Flutter).
4. Firebase proje kurulumu ve temel authentication akışını başlat.
5. Yönetici panelini web app olarak ilk sürüme al.
